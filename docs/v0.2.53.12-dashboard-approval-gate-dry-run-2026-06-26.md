# v0.2.53.12 Dashboard ApprovalGate dry-run 按钮联调(2026-06-26)

> **状态**:✅ 已落地 · **范围**:HTML dry-run 按钮 + inspector 拒写原因展示 · **边界**:不写 DB / 不发 SMTP / 不接 business writer

## 结论

Mail / Notes / Finance 队列项新增 dry-run 按钮,点击后 `POST /api/approval-gate/actions`( `dry_run: true` ),在右侧 inspector 展示拒写原因与审计预览。API 离线时使用静态兜底文案,不发起 POST。

## 前端行为

| 页面 | 按钮 | action | target_id |
|------|------|--------|-----------|
| Mail · pending_send | dry-run 审批 | `outbox.approve` | `outbox_id` |
| Mail · 其他状态 | dry-run 取消 | `outbox.cancel` | `outbox_id` |
| Notes | dry-run 确认 | `notes.confirm` | `apple_note_id` |
| Finance | dry-run 忽略 | `finance.dismiss_anomaly` | `date\|counterparty\|amount` |

## 预期响应(默认禁写)

| 场景 | HTTP | error |
|------|------|-------|
| API 在线 · 未设 `DASHBOARD_WRITE_API=1` | 403 | `write_disabled` |
| API 离线 | — | `api_offline`(静态兜底) |

所有路径保证 `write_executed=false`。
