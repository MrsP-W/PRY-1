## 📂 缺 2/3 wrapper 应创建内容(预览 · 沿 install 脚本 §3 heredoc)

### `~/bin/my-ai-employee-imap-sync`(应建)

```bash
#!/usr/bin/env bash
# 部署于 2026-07-08 <time> by scripts/launchd_install.sh
ENV_FILE="<PROJECT_ROOT>/.env"
IMAP_USER=""
if [[ -f "${ENV_FILE}" ]]; then
  IMAP_USER=$(grep -E '^IMAP_USER=' "${ENV_FILE}" | head -1 | cut -d= -f2- | tr -d '"' | tr -d "'")
fi
if [[ -z "${IMAP_USER}" ]]; then
  echo 'IMAP_USER not set in .env' >&2
  exit 2
fi
exec uv run --project "<PROJECT_ROOT>" python scripts/sync_imap.py sync --provider qq --email "${IMAP_USER}"
```

**红线**:
- ❌ 不写真凭据(只读 IMAP_USER,SMTP 密码走 Keychain)
- ❌ 需先确认 `.env` 中 `IMAP_USER` 已设置(否则 IMAP wrapper 启动会 exit 2)

### `~/bin/my-ai-employee-start`(应建)

```bash
#!/usr/bin/env bash
# 部署于 2026-07-08 <time> by scripts/launchd_install.sh
exec bash "<PROJECT_ROOT>/ops/start-digital-employee.sh" start
```

**红线**:
- ❌ 撞坑 #81 实战暴露:数字员工启动会触发 TCC Python.framework 3.12 授权弹窗,未授权时 fail 噪声
- ❌ RunAtLoad=true → 登录时自动启动,误启可能导致 fail 日志堆积
