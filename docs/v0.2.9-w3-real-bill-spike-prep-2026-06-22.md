### 风险门控 1:`WECHAT_REAL_IMPORT=1` env 门控

**关键代码**(沿 D6.6 范本,`scripts/spike_import_wechat_real.py`):
```python
_WECHAT_REAL_IMPORT_ENV: str = "WECHAT_REAL_IMPORT"
_WECHAT_REAL_IMPORT_VALUE: str = "1"

# 默认 os.environ.get("WECHAT_REAL_IMPORT") != "1" → 抛错,绝不真导入
if os.environ.get(_WECHAT_REAL_IMPORT_ENV) != _WECHAT_REAL_IMPORT_VALUE:
    raise RuntimeError(

        f"实际 env[{_WECHAT_REAL_IMPORT_ENV}]={os.environ.get(_WECHAT_REAL_IMPORT_ENV)!r}。"
    )
```

**核对结论**:✅ **`WECHAT_REAL_IMPORT=1` env 门控就绪**,默认安全,必须显式设置才允许真导入。

### 风险门控 4:CSV 文件名校验 + 列名嗅探

**关键代码**(沿 D6.1 `detect_version`):
```python
def detect_version(csv_path: str) -> str:
    """嗅探 CSV 版本(2024/2025/2026)。"""
    with open(csv_path, "r", encoding="utf-8") as f:
        first_line = f.readline()
    if "交易时间" in first_line and "金额(元)" in first_line:
        return "2024"
    elif "日期" in first_line and "收/付" in first_line:
        return "2025"
    elif "时间" in first_line and "金额" in first_line:
        return "2026"
    else:
        raise ValueError(f"无法识别 CSV 版本: {csv_path}")
```

**核对结论**:✅ **列名嗅探就绪**,未匹配抛 `ValueError`,防误解析。
