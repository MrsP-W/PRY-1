## 4. 沿 D6.6 4 重防误发范本(详细)

### 4.1 WECHAT_REAL_IMPORT=1 env 门控

```python
import os
if os.environ.get('WECHAT_REAL_IMPORT') != '1':
    print("❌ WECHAT_REAL_IMPORT env 门控未设", file=sys.stderr)
    return 1
```

**设计目的**: 真实 CSV 写入 transactions 表是破坏性操作,必须显式确认。

### 4.2 --csv-path 必传真实 CSV

```python
csv_path = Path(args.csv_path)
if not csv_path.is_file():
    print(f"❌ CSV 文件不存在: {csv_path}", file=sys.stderr)
    return 1
if "faker" in csv_path.name.lower():
    print(f"❌ 拒绝 faker CSV: {csv_path}", file=sys.stderr)
    return 1  # 防误传 faker 样本
```

### 4.3 --max-rows 1 限制单笔

```python
if args.max_rows != 1:
    print(f"❌ --max-rows 必须为 1(本轮 spike 仅支持 1 笔)", file=sys.stderr)
    return 1
```

**设计目的**: 单笔 spike 防误传大文件(沿 D5.6.5 4 重防误发范本)。

### 4.4 --confirm 文本完全匹配

```python
_REQUIRED_CONFIRM = "yes-i-understand-this-imports-real-bill"
if args.confirm != _REQUIRED_CONFIRM:
    print(f"❌ --confirm 必须为 '{_REQUIRED_CONFIRM}'", file=sys.stderr)
    return 1
```

**设计目的**: 用户必须**主动**输入确认文本,防止脚本误触发。
