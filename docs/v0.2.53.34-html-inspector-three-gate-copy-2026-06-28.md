## 2. 设计:`THREE_GATE_COPY` 集中文案

### 2.1 常量结构

```javascript
THREE_GATE_COPY = {
  first: { open, closed },
  second: { confirm_required_per_action },
  third: { open, closed, env_only },
  outcome: { disabled, needs_confirmation, writer_required, dry_run_ready, api_offline, unknown },
  writerBadge: { env, impl, ready },
  offlineDryRun: { error, reason, required },
}
FALLBACK_DRY_RUN_STATUS = { outcome: "disabled", first_gate: "closed", ... }
```

### 2.2 渲染入口

| 函数 | 职责 |
|------|------|
| `renderDryRunThreeGate()` | 同步 header / system / inspector 四门 |
| `renderOfflineApprovalState()` | API catch 路径;`offline: true` → outcome api_offline |
| `resolveOutcomeMeta()` | writer_required 细分 env-only vs env-off |
| `submitApprovalDryRun()` | 离线 payload 带三门 false + 统一 reason |
