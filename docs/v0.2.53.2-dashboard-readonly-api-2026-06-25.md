## 端点

| 路径 | 说明 |
|------|------|
| `GET /api/status` | 质量门快照、Git HEAD、Keychain present/missing、ApprovalGate 状态 |
| `GET /api/tasks/today` | 今日待办(邮件草稿 + Notes待确认 + 财务异常) |
| `GET /api/outbox` | 邮件草稿队列(只读) |
| `GET /api/notes/pending` | Notes 待确认列表 |
| `GET /api/finance/anomalies` | 财务异常列表 |
| `GET /health` | 存活探针 |
