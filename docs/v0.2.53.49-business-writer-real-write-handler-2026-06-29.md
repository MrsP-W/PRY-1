## 2. 设计决策(2026-06-29 锁定)

### 2.1 写保护锁 `_real_write_handler_enabled`

```python
def __init__(
    self,
    *,
    outbox_store: OutboxStore | None = None,
    real_write_handler_enabled: bool = False,  # ← 写保护锁(默认锁定)
) -> None:
    ...
    self._real_write_handler_enabled = real_write_handler_enabled

def _check_write_protection(self) -> None:
    if not self._real_write_handler_enabled:
        raise NotImplementedError(
            "BusinessWriterImpl 写保护锁未开(_real_write_handler_enabled=False);"
            "路径 4 实际写入留 v0.2.53.19 handler 启用 + 8/1 后 + 用户明确授权"
        )
```

**写保护锁 3 层语义**:
| 状态 | 行为 | 用途 |
|------|------|------|
| `False`(默认) | 4 动作 raise NotImplementedError | 生产环境(v0.2.53.19 未启用前) |
| `True` + 无依赖 | `_check_dep` raise | 测试场景(依赖未注入) |
| `True` + 有依赖 | 真实调 service(写 DB) | 测试场景(撞坑 #18 风险门控) |

### 2.2 4 动作方法统一骨架(升级 v0.2.53.46)

```python
def approve_outbox(self, target_id: str, *, audit: AuditContext) -> WriteResult:
    self._check_dep(self._outbox_store, "outbox_store")
    err = self._validate_target_id(target_id)
    if err is not None:
        return WriteResult(success=False, affected_id=None, error="invalid_target_id", reason=err)
    self._check_write_protection()                              # ← v0.2.53.49 新增
    return self._call_service_approve_outbox(target_id, audit=audit)  # ← v0.2.53.49 新增
```

**4 动作公共模板**(approve_outbox / cancel_outbox / confirm_note / dismiss_anomaly):
- `_check_dep`:依赖检查(撞坑 #18 风险门控)
- `_validate_target_id`:参数校验(严判 type + strip + 非空)
- `_check_write_protection`:写保护锁(撞坑 #18 默认锁定)
- `_call_service_xxx`:真实调 service(仅测试场景)
