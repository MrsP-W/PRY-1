### 2.3 撞坑 #70 范本(供未来参考)

```python
# ❌ 错误:中文注释 + # type: ignore[...] 同行,mypy 不识别
tone=123,  # 非法 type, 运行时严判拦截  # type: ignore[arg-type]

# ✅ 正确:# type: ignore[...] 必须在前(在第一个 # 之后)
tone=123,  # type: ignore[arg-type]  # 非法 type, 运行时严判拦截
```

**根因**:Python 解析器把第一个 `#` 之后到行尾都作为注释,mypy 在解析时遇到多个 `#` 注释会丢失 type:ignore 上下文。**正确范本**:type:ignore 注释必须紧跟代码,中文说明注释放最后。
