### 新增 S10 SMTP 多 provider(沿 B outlook/gmail 子阶段)

**目标**:v0.2 阶段从单一 smtp.qq.com provider 扩展到 outlook.com / gmail.com 多 provider,沿 `connectors/smtp.py` 拆分 base + outlook + gmail + provider 工厂范本。

**关键决策**:
- provider 工厂模式: `SmtpProviderFactory.create(provider_name) -> BaseSmtpTransport`
- 4 重防误发精神(沿 D5.6.5)不变
- 环境变量:`SMTP_PROVIDER=qq|outlook|gmail` 显式选 provider
- 退出码:沿 import_wechat.py 4 退出码(0/1/2/3)

**测试范本**:`tests/scripts/test_smtp_provider_factory.py` + `test_outlook_transport.py` + `test_gmail_transport.py`(3 套独立 provider 单元测试)

### 新增 S11 智能财务异常检测(沿 D8 子阶段)

**目标**:LLM 异常交易检测(消费金额异常 / 商家画像漂移 / 频率异常)+ 商家画像 800 → 1500 扩 + 异常告警接入菜单栏。

**关键决策**:
- 复用 `ai/categorizer.py` 范本(关键词 → LLM 双层分类)
- 异常规则 3 类:消费金额异常(>3σ)+ 商家画像漂移 + 频率异常(>5 笔/小时)
- 菜单栏弹通知(macOS Sequoia Notification Center 范本)
- 退出码:沿 import_wechat.py 4 退出码

**测试范本**:`tests/ai/test_anomaly_detector.py` + `tests/finance/test_merchant_profile_drift.py`
