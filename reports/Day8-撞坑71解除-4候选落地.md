**改动范围**:
- 新建 `src/my_ai_employee/core/notes_encryption.py`(`NotesFieldCipher` 字段配置 + `NotesCipher` Protocol + `NotesCipherStub` 默认明文透传 + `NotesCipherImpl` HMAC-SHA256 链 10000 次 + 随机 IV + XOR 流密码 + 密文前缀 `enc:v1:` + `ENABLE_NOTES_ENCRYPTION` opt-in + `build_notes_cipher` 工厂)
- `tests/core/test_notes_encryption.py` 新增 38 测试(Stub 7 + Impl 12 + opt-in 13 + 字段 4 + 稳定性 2)

**关键技巧**:
- **撞坑 #65 opt-in 4 阶段**:`ENABLE_NOTES_ENCRYPTION=0` 默认 Stub → `=1` 启用 → 缺 master_key 降级 → 短 key(<16 字节)再降级
- **Protocol 不声明公共属性**:`is_runtime_impl` / `name` 由具体类提供,避免 frozen dataclass setter 类型冲突(Protocol 抽象方法 + dataclass 字段冲突)
- **HMAC-SHA256 链 10000 次**派生 key(类似 PBKDF2 但纯 stdlib 无 cryptography 依赖)
- **随机 IV**:`os.urandom(16)` 每次 encrypt 新 IV,同明文 + 同 key → 密文每次不同
- **字段级加密**:`body` / `title` 字段加密,`id` / `status` / `created_at` 不加密(便于查询)
- **撞坑 #64 公共 API 一致性**:`_CIPHERTEXT_PREFIX_V1 = "enc:v1:"` 版本化前缀
